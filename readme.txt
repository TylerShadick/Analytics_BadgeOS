== Description ==

[BadgeOS Analytics](https://github.com/TylerShadick/Analytics_BadgeOS "BadgeOS Analytics on GitHub")&trade; adds onto BadgeOS by allowing you to anonymously track user page and learning actvity engagement in Google's Universal Analytics. 

Features of BadgeOS Analytics:

**Track Submissions, Nominations and Unlocked Achievements**

*   Tracks submissions/nomination => provides information as to which page they submitted/nominated on
*   Tracks when someone unlocks an achievement => provides achievement type and achievement name

**Optionally use Universal Analytics to Track Pageviews**

*   Provides a field where you can paste in your Universal Analytics tracking code
*   Optionially ignore tracking pageviews for administrators (does not ignore learning events).

Setting up BadgeOS Analytics

*   Install the plugin
*   If you haven't already, set up a Universal Analytics account (different from a standard Google Analytics account) on [Google's Analytics](http://www.google.com/analytics/ "Google Analytics Homepage")
*   Copy your Universal Analytics Account ID (UA-XXXXXXXX-Y) into the Google Analytics Settings page under the Settings tab on your Wordpress admin page
*   Optionally copy your UA Tracking Code into the tracking field
*   Optionally choose to track or ignore Administrator pageviews
*   Begin tracking BadgeOS learning activity and engagement on Google Analytics
